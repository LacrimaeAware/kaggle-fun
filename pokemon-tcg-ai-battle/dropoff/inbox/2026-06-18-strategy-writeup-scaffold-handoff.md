## Main framing

Do not write as if the project is finished.

As of 2026-06-18, this competition is only a few days old. The write-up should preserve the research process while the agent is still changing.

The current best narrative is:

```text

```

## Important process rule

For every experiment from now on, append a concise write-up note with:

- hypothesis
- implementation
- dataset size / game count
- metric
- result
- interpretation
- whether it changed the submitted agent
- one figure/table candidate

This makes the Strategy Category report a byproduct of the research rather than a rushed after-the-fact reconstruction.
