- Methodology: mostly things already known and acted on. The 0.75 PH-fix over-read is already
  retracted (selfbase=0.567 at n=120, see `pokemon-tcg-search-status` memory + commit 5671b0b).
  The genuinely new one is M2: the registry has gone write-only (frozen at R009 while
  phfix/N32/v3 results live only in loose `docs/workstreams/*.json`; no code reads
  `registry.py`; the anti-rehash gate is unenforced).

  diverged into 3 thresholds. You could not score one heuristic on a fixed state without
  running full games, and the never-throw/never-timeout rule had zero test coverage.
